Deploying a Fall Detection Program Using Amazon SageMaker

About 36 million older adults fall each yearâ€”resulting in more than 32,000 deaths. Many of these falls take days to recover from and are not responded to immediately. The earlier a fall is responded to, the quicker the person can heal. A fall detection program can make sure that falls are responded to quickly.

How It Works

Once the program first starts up and initializes, there is an option to use a camera or a video. If you choose the camera option, then it will turn the camera the computer is connected to on and run the program on the camera live stream. If you choose the video option, it will ask you to input the video's path and run the program on the desired video. After choosing a source for the program, it will recognize the person in the source and put a bounding box around him/her. A bounding box is just a rectangle that covers a selected part of an image or video. Then the program establishes a ratio of the height to width of the bounding box. If a person falls, then the ratio of the bounding box will reverse and change. The program will then take a screenshot of the fallen person and notify a appropriate personel via SMS or Email.

Applications

This fall detection program can have personal and commercial applications. The personal applications of the fall detection program can be in a person's home. If a person at home has a risk of falling, it would be beneficial to have this fall detection program. The program would monitor the person and quickly notify a family member if they fall. The commercial applications of the fall detection program are mainly in nursing homes and rehabilitation centers. If an older adult falls in a nursing home, the fall detection program can notify the nurse, so the situation is attended to quickly.

This program is currently downloadable but has many limitations. Using Amazon SageMaker, I am hoping to bring this online so it accessible anywhere, by anybody. SageMaker is a segment of the well-known Amazon Web Services (AWS). AWS has many segments for everything from IoT to Blockchain. SageMaker is a segment that deals mainly with Machine Learning. SageMaker helps developers prepare, build, train, and deploy machine learning models quickly by bringing together a broad set of capabilities. Using the myriad of features on SageMaker and other AWS services, I hope to deploy the fall detection program online.
